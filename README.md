# 🧠 PyTorch POM Project

A modular, testable deep‑learning project built with **PyTorch** using the **Page Object Model (POM)** architecture.  
This structure cleanly separates data loading, model definition, training logic, and evaluation into independent modules — making the project easy to maintain, extend, and test.

---

Run training + evaluation:
python main.py

Running Tests:
pytest

Why POM for Machine Learning?

Using the Page Object Model pattern in ML projects provides:

Clear separation of concerns

Easier debugging

Reusable components

Cleaner test structure

Professional architecture for real‑world projects
